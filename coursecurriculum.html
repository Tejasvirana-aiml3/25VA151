WD WORKSHOP COURSE CURRICULUM

Course Objective

* Introduce students to the fundamentals of web technologies and the full-stack development environment. Familiarize students with modern tools like VS Code and Git for development and version control practices.
* Provide hands-on experience with HTML, CSS, and GitHub for collaborative development. Develop practical skills in creating structured, styled, and interactive web pages.

At the end of this course, student will be able to:

  • Demonstrate understanding of basic web technologies and client-server architecture.
  • Create well-structured web pages using HTML elements, forms, and multimedia integration.
  • Use Git and GitHub for version control and collaborative web development.
  • Apply CSS for styling, layout design, and responsive formatting

Syllabus

  1. UNIT-01 -- Introduction to Web Technology
  2. UNIT-02 -- Web Page Development using HTML
  3. UNIT-03 -- Version Control with Git & GitHub and CSS Fundamentals
  4. UNIT-04 -- CSS for Advanced Styling and Responsive layout
